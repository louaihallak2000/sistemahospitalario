📋 ¿QUÉ ES ESTE SISTEMA?
────────────────────────
Un sistema completo de gestión hospitalaria para emergencias con:
• Gestión de pacientes y episodios
• Sistema de triaje por colores
• Panel de enfermería avanzado
• Historias clínicas automáticas
• Notificaciones en tiempo real (WebSocket)
• Interfaz web moderna y responsive

================================================================================
🚀 INICIO RÁPIDO (3 PASOS)
================================================================================

PASO 1: REQUISITOS
───────────────────
✅ Windows 10/11
✅ Python 3.8+ instalado (https://python.org)
✅ Node.js 16+ instalado (https://nodejs.org)
✅ Conexión a internet (para descargar dependencias)

PASO 2: INICIAR SISTEMA
────────────────────────
1. Haz doble clic en: "INICIAR_SISTEMA_HOSPITALARIO.bat"
2. Espera a que aparezcan 2 ventanas (Backend + Frontend)
3. El navegador se abrirá automáticamente en http://localhost:3000

PASO 3: ¡USAR EL SISTEMA!
──────────────────────────
• El sistema ya tiene datos de ejemplo cargados
• 15 pacientes y 20 episodios listos para usar
• Usuarios de prueba creados automáticamente

⚠️ PROBLEMAS COMUNES Y SOLUCIONES
================================================================================

PROBLEMA: "No se encuentra Python"
──────────────────────────────────
SOLUCIÓN: 
1. Instala Python desde https://python.org
2. Asegúrate de marcar "Add to PATH"
3. Reinicia la computadora

PROBLEMA: "No se encuentra Node.js"
───────────────────────────────────
SOLUCIÓN:
1. Instala Node.js desde https://nodejs.org
2. Usa la versión LTS (recomendada)
3. Reinicia la computadora

PROBLEMA: Error "Puerto ya en uso"
──────────────────────────────────
SOLUCIÓN:
1. Ejecuta "DETENER_SISTEMA_HOSPITALARIO.bat"
2. Espera 10 segundos
3. Vuelve a iniciar el sistema

PROBLEMA: Frontend no carga
────────────────────────────
SOLUCIÓN:
1. Verifica que el Backend esté funcionando
2. Ve a http://localhost:8000/docs
3. Si no funciona, reinicia el sistema completo
